# Tailwind CSS Playground & Template Library

This repository is a **personal Tailwind CSS learning playground** and **template reference library**.

The goal of this project is to:
- Learn Tailwind CSS through hands-on styling
- Build reusable UI templates (cards, layouts, forms, etc.)
- Maintain a central place to experiment with general styles
- Easily reuse templates in future projects (Laravel, plain HTML, or others)
